Stylized PBR Barrel — PBR in Blender (High-level)

Lesson packet for: Intermediate 3D Animation / Game Art

Learning Objectives

  • Understand the PBR workflow (albedo, roughness, metallic, normal, AO).
  • Create a low-to-mid poly stylized barrel in Blender suitable for real-time engines.
  • Bake and assemble texture maps and export optimized assets for a game.
  • Apply design constraints for style consistency and performance (texture atlases, LODs).

Lesson Tasks (Instructor-led + Studio Work)

  1. Reference & thumbnail — gather 3 visual references of stylized barrels and make 3 thumbnails (silhouette-focused).
  2. Blockout mesh in Blender; maintain silhouette and silhouette-readability at game scale.
  3. UV unwrap with texel density target (e.g., 4–8 px/cm equivalent) and pack for atlas use.
  4. High-to-low workflow: sculpt or bevel details, bake normal/AO to low poly.
  5. Create PBR materials in Blender's Principled BSDF and export maps (png, power-of-two sizes).
  6. Integrate in a simple runtime preview (Unity/Unreal/PlayCanvas) or deliver as FBX + maps.

High-level Discussion Questions

Q1: What visual cues make a stylized object readable at small sizes, and how does that influence modeling choices?
Q2: In a PBR pipeline, when and why would you intentionally break physical accuracy for stylization?
Q3: How do texture atlases and LODs trade off between memory and visual fidelity? Provide a concrete example for this barrel.
Q4: Describe three strategies to reduce shader cost on mobile platforms for a PBR material.
Q5: How does the choice of metallic vs. specular workflow affect your base color (albedo) painting strategy?
